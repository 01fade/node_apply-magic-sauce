# Using Node and Apply Magic Sauce API

Research Paper: http://www.pnas.org/content/110/15/5802.full.pdf?with-ds=yes

##Setup

- ```$ npm install```
- rename ```secrets_example.json``` to ```secrets.json```
- paste ```customer_id``` and ```api_key```
    - leave ```token``` for now
- ```$ node app.js```
